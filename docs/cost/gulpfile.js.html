Metric Value Description
Maintainability 62.44 This is the relative maintainability of the file on a scale of 0-100. Higher is better.
Difficulty 4.25 This is a measure of how difficult it is to understand the code. Lower is better.
Cyclomatic Complexity 1 This is how complex the code is based on the number of paths through the code. Lower is better
Cyclomatic Density 8.33 This is a re-expression of the above metric as a percentage based on the lines of code. Lower is better.
Effort 1105.28 This is the amount of effort needed to write the code based on it's difficulty and length
Time 0.02 Hours This is the estimated amount of time needed to write the file based on the effort required.
Cost $0.80 This is the estimated cost to develop the file based on the time needed to develop it.
Bugs 0.09 This is the estimated number of bugs in the file.
Logical SLOC 12 This is the number of lines of software that are considered logic.
Physical SLOC 16 This is the total lines of source code including up to 25% whitespace, but ignoring excessive formatting.
